# Risorse per lo sviluppo

Questa sezione contiene la lista delle applicazioni software e delle librerie utilizzate durante il corso. Inoltre, presenta alcune risorse aggiuntive che possono essere utili per la creazione di applicazioni web.

## Software in uso

- [Node.js](https://nodejs.org/en/) 22.x (LTS) e npm
- [Visual Studio Code](https://code.visualstudio.com/)
- Google Chrome o Mozilla Firefox (versioni recenti)
- React Developer Tools ([estensione Chrome](https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi?hl=en) o [estensione Firefox](https://addons.mozilla.org/en-US/firefox/addon/react-devtools/))
- [DB Browser for SQLite](https://sqlitebrowser.org/) oppure [SQLite Viewer](https://marketplace.visualstudio.com/items?itemName=qwtel.sqlite-viewer) per Visual Studio Code
- [git](https://git-scm.com/)

Installeremo poi le altre librerie necessarie (per es., React ed Express) tramite npm.

## Documentazione e guide

- Sito fondamentale per documentazione su tecnologie e sviluppo Web: [Mozilla Developer Network](https://developer.mozilla.org/) (MDN)
- [React](https://react.dev)
- [Bootstrap 5.3](https://getbootstrap.com/docs/5.3/getting-started/introduction/) and [React Bootstrap](https://react-bootstrap.github.io/)
- Per approfondire i segreti di JavaScript: [You Don't Know JS Yet](https://github.com/getify/You-Dont-Know-JS) (2nd edition)
- [The Modern JavaScript Tutorial](https://javascript.info/)

## Altre risorse utili

- Simulare un dispositivo mobile da browser desktop, per esempio con il [Device Mode](https://developer.chrome.com/docs/devtools/device-mode) di Chrome
- Web design:
  - [Google Fonts](https://fonts.google.com/) è un'ottima risorsa per font pronti all'uso. [FontPair](https://fontpair.co/) può aiutare ad abbinarli.
  - [Unsplash](https://unsplash.com/) è un ottimo punto di partenza per cercare foto e immagini campione con licenza Creative Commons.
  - [Adobe Color](https://color.adobe.com/) è un'applicazione web interattiva per aiutare a scegliere schemi di colori che stanno bene insieme.
